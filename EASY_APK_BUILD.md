# Build APK - Easiest Method

## RECOMMENDED: EAS Build (Cloud Build)

No Android SDK needed! Build in the cloud.

### Step 1: Install EAS CLI
```bash
npm install -g eas-cli
```

### Step 2: Login to Expo
```bash
eas login
```
Create free account at expo.dev if needed

### Step 3: Build APK
```bash
eas build --platform android --profile preview
```

**What happens:**
- ✅ Builds in Expo's cloud servers
- ✅ Takes 10-15 minutes
- ✅ Gives you download link
- ✅ No local setup needed!

### Step 4: Install on S24 Ultra
- Download APK from the link
- Transfer to phone
- Install

---

## That's it! Three commands total.
